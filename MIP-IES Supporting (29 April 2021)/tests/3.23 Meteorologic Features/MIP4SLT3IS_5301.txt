Test description

Test IDMIP4SLT3IS_5301Test TitleCreate Atmosphere.Execution Priority1ObjectiveBSO validation. Test whether the Producer/Provider is able to create an Atmosphere (MeteorologicFeature) and the Consumers/Receivers are able to understand (and display) it correctly.ScenarioThe Producer creates an Atmosphere (MeteorologicFeature)
The Consumers receive the information and interpret it correctly.EnvironmentInternet or Co-located. Exchange Pattern: R/R, P/S.Participation2 or more.MTRSN/APre-test ConditionsThe Producer/Provider has created a topic which includes message elements belonging to the MeteorologicFeature message schema area.
The Consumers have subscribed to this topic.Test InputsN/A.ConclusionThis test is considered a success if the Atmosphere is processed and interpreted correctly on all Nations' C2IS.Test OutputsN/A.TraceabilityREQ_INF_0002

* InversionLayer
* PressureMeasure
* PressureSystemCategory
* Temperature
* TemperatureGradient
