Test description

Test IDMIP4SLT3IS_5271Test TitleMultiple ControlFeature ControlFeatureSecurityStatusCode update.Execution Priority1ObjectiveBSO validation. Verify the ability to update the ControlFeatureSecurityStatusCode of a ControlFeature in a nation's C2IS.ScenarioNation A creates a ControlFeature with ControlFeatureSecurityStatusCode = None. 
Nation B receives the information and interprets it correctly.
Nation A updates the ControlFeature to ControlFeatureSecurityStatusCode = Guarded.
Nation B receives the information and interprets it correctly.
Nation A updates the ControlFeature to ControlFeatureSecurityStatusCode = Secured.
Nation B receives the information and interprets it correctly.EnvironmentInternet or Co-located. Exchange Pattern: R/R, P/S.Participation2MTRSApplicablePre-test ConditionsNation A has created a Topic which includes message elements belonging in the ControlFeature message schema area, for instance Ground Picture > Battlespace Geometry. 
Nation B has subscribed to this Topic.Test InputsN/A.ConclusionThis test is considered a success if the updated ControlFeature is interpreted and displayed correctly on both Nations' C2IS.Test OutputsN/A.TraceabilityREQ_INF_0003.
