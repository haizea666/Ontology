Test description

Test IDMIP4SLT3IS_5604Test TitleCBRNEquipment Sub-Type update.Execution Priority1ObjectiveBSO validation. Test whether the Producer/Provider is able to create and update a CBRNEquipment Sub-Type and the Consumers/Receivers are able to understand (and display) it correctly.ScenarioThe Producer creates a CBRNEquipment Sub-Type with values for all attributes.
The Consumers receive the information and interpret it correctly.
The Producer updates this CBRNEquipment with new values for all attributes, specifically for the Sub-Type.
The Consumers receive the information and interpret it correctly.EnvironmentInternet or Co-located. Exchange Pattern: R/R, P/S.Participation2 or more.MTRSApplicablePre-test ConditionsThe Producer/Provider has created a topic which includes message elements belonging to the Equipment message schema area.
The Consumers have subscribed to this topic.Test InputsN/A.ConclusionThis test is considered a success if the CBRN Equipment is processed and interpreted correctly on all Nations' C2IS.Test OutputsN/A.TraceabilityREQ_INF_0002, REQ_INF_0003
